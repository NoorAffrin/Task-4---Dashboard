Task 4 – Dashboard Design (Tableau)

This project uses Tableau to analyze the Sales Dataset and build an interactive dashboard that highlights key sales and profit insights for decision-making.

Key Insights

- Total Sales, Total Profit, Average Order Value
- Monthly Sales Trends
- Top Products / Categories
- Sales vs Profit relationship
- Actionable recommendations based on performance patterns

Tableau Dashboard Features

- Line Chart: Sales Over Time
- Bar Chart: Top Products by Sales
- Scatter Plot: Sales vs Profit
- Filters: Month, Product, Category
- Interactive tooltips and drill-down capability

How to Use

1. Open `Sales_Dashboard.twbx` in Tableau Desktop (or Tableau Public).
2. Interact with filters and explore insights.
3. Review summary KPIs and highlighted trends.

Included Files

- Sales Dataset.csv
- Sales_Dashboard.twbx (Packaged Tableau workbook)
- charts/ (PNG images exported from Tableau)
- README.md

Tools Used

Tableau • Excel/CSV • PowerPoint
